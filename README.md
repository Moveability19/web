# MovEability Website

Structured multi-page static website for the MovEability pediatric physiotherapy clinic.

## Pages

- `index.html` - Homepage and quick navigation hub
- `about.html` - Clinic approach, testimonials, and founder
- `services.html` - Services, workflow, and programs
- `conditions.html` - Expandable condition categories
- `contact.html` - Contact details and WhatsApp appointment form

## Assets

- Shared styles live in `css/styles.css`
- Shared behavior lives in `js/script.js`
- Local video files live in `assets/videos/`
- `assets/images/` is ready for future local image assets

## Notes

- Open `index.html` in a browser to preview the site.
- Navigation and styling are centralized so future edits only need to be made once.
